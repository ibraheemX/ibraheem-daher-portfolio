About

This project is a Django-based task management application. Users can manage their daily tasks by creating, editing and deleting tasks. The application focuses on backend logic, database operations and user authentication.

Features

  • ✔ User Authentication
  • ✔ Create Tasks
  • ✔ Update Tasks
  • ✔ Delete Tasks
  • ✔ Task Management
  • ✔ Database Integration
